## 🚀 How to Run Any Project

1. Open **VS Code**
2. Open the project folder
3. Open the `.py` file
4. Press `Ctrl + `` ` to open terminal
5. Type: `python filename.py` and press **Enter**
6. Answer the questions one by one
7. See your beautiful output! 🎉
