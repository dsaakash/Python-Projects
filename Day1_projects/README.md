## 🚀 How to Run Any Project

```bash
# Step 1: Navigate to the project folder
cd Day1/Level_1/01_My_Profile_Card

# Step 2: Run the Python file
python my_profile_card.py

# Step 3: Answer the questions and see the output!
```

Or in **VS Code**: Open the `.py` file → Press `F5` or `Ctrl+F1` to run.
